# SafeRestore

Minimal Vite + React + Tailwind app with a small Node/Express AI chat server.

## Setup
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview (client only)
```bash
npm run preview
```

## Environment
Create a `.env` file from `.env.example`:
```
OPENAI_API_KEY=your_key_here
```
